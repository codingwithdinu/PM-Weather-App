# 🌤️ Weather App (Full Stack)

## 🚀 Features
- Get real-time weather data
- 5-day forecast
- Store searched data in MongoDB
- View saved weather history

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB Atlas
- API: OpenWeatherMap

## 📦 How to Run

### Frontend
Open index.html in browser

### Backend
cd backend
npm install
node server.js

## 🌐 API Endpoints
- GET /api/weather/all
- POST /api/weather/save
